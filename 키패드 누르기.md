키패드 누르기

점과 점 사이 거리 공식과 튜플 언패킹을 이용하여 풀어보았다.

tuple unpacking  

ex.) a, b = (0, 1)

a = 0 , b =1

문제 출처 : https://programmers.co.kr/learn/courses/30/lessons/67256
    

```python
def solution(numbers, hand):
    l,r = '*', '#' #초기값 설정
    anwser = '' 

def dt(now, num) : #numbers in [2,5,8,0]일 때 거리구하는 함수
    pad = {1:(0,0), 2:(0,1), 3:(0,2),
          4:(1,0), 5:(1,1), 6:(1,2),
          7:(2,0), 8:(2,1), 9:(2,2),
          '*':(3,0), 0:(3,1), '#':(3,2)}

    now_x, now_y = pad[now]
    num_x, num_y = pad[num]

    return abs(num_x-now_x)+abs(num_x-now_y)
    '''점과 점 사이의 거리
    절댓값 함수를 사용하지 않고 (num_x-now_x)**2 + abs(num_x-now_y)**2
    라고 하여도 같은 결과가 나온다.'''
    

for i in numbers : #왼쪽 열
    if i in [1,4,7] :
        anwser+='L'
        l = i
    elif i in [3,6,9] : #오른쪽 열
        anwser+='R'
        r = i
    elif i in [2,5,8,0] : #가운데 열
        l_dt = dt(l, i)
        r_dt = dt(r, i)
        if l_dt < r_dt :
            anwser += 'L'
            l = i
        elif l_dt == r_dt : #거리가 같을 경우
            if hand == 'right' : #오른손잡이
                anwser+='R'
                r = i
            else : #왼손잡이
                anwser+='L'
                l = i
        else :
            anwser+='R'
            r = i
return anwser
```
